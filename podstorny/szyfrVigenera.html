Szyfr Vigenère'a

Szyfr Vigenère'a to polialfabetyczny szyfr przestawieniowy, który używa kilku różnych alfabetów do zaszyfrowania tekstu. Polega na użyciu jednego klucza, który składa się z wielu liter. Klucz ten jest powtarzany cyklicznie do momentu, w którym długość klucza jest równa długości tekstu jawnego. Następnie każda litera tekstu jawnego jest szyfrowana przy użyciu litery klucza, a następnie zastosowanie odpowiedniego alfabetu, zależnego od pozycji litery klucza. W skrócie, szyfr Vigenère'a oferuje zwiększoną ochronę przed atakami z użyciem analizy częstotliwości i wymaga użycia wielu różnych alfabetów, ale nie jest bezpieczny przed zaawansowanymi atakami kryptoanalitycznymi

Plusy szyfru Vigenère'a:

  • Jest trudny do złamania przez ataki z użyciem analizy częstotliwości, ponieważ każda litera tekstu jawnego jest szyfrowana inną literą klucza
  • Zwiększa złożoność szyfru, ponieważ wymaga wielokrotnego użycia różnych alfabetów
  • Łatwość w użyciu i szyfrowanie tekstu jest szybkie

Minusy szyfru Vigenère'a:

  • Szyfr Vigenère'a może być złamany za pomocą ataków z użyciem kryptoanalizy, takich jak analiza częstotliwości liter klucza
  • Jest podatny na ataki z użyciem metody Kasiski, która wykorzystuje powtarzające się ciągi liter w tekście zaszyfrowanym, aby odkryć długość klucza
  • Szyfr Vigenère'a nie ukrywa długości tekstu jawnego, co ułatwia atakującemu odkrycie klucza i złamanie szyfru

        if (c >= 'A' && c <= 'Z') {
            char kluczChar = toupper(klucz[kluczIndex % klucz.length()]);
            int kluczOffset = kluczChar - 'A';

        }
    }
    return szyfr;
