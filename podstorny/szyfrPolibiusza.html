Szyfr Polibiusza

Szyfr Polibiusza to szyfr polegający na zastępowaniu liter tekstu jawnego przez ich odpowiedniki w tabeli Polibiusza. Tabela ta składa się z dwóch wierszy i dwóch kolumn, w których litery alfabetu są zapisywane w parach. Każda para liter jest zastępowana przez dwucyfrowy kod, który odpowiada pozycji pary w tabeli. Przykładowo, litera "A" jest zastępowana przez "11", litera "B" przez "12", a litera "Z" przez "26". Tekst jasny jest następnie zapisywany w postaci dwucyfrowych liczb, co umożliwia szyfrowanie tekstu przy użyciu prostych operacji matematycznych. W skrócie, szyfr Polibiusza jest łatwy w zastosowaniu i oferuje dobre zabezpieczenie przed analizą częstotliwości liter, ale nie jest bezpieczny przed zaawansowanymi atakami kryptoanalitycznymi

Plusy szyfru Polibiusza:

  • Łatwy w zastosowaniu i zrozumieniu
  • Zapewnia dobre zabezpieczenie przed analizą częstotliwości liter, ponieważ każda litera jest zastępowana przez dwucyfrowy kod
  • Szyfrowanie i deszyfrowanie tekstu jest łatwe i szybkie

Minusy szyfru Polibiusza:

  • Szyfr Polibiusza nie jest bezpieczny przed atakami z użyciem analizy częstotliwości dwucyfrowych kodów, co może ujawnić wzorce w tekście
  • Szyfr Polibiusza nie ukrywa długości wiadomości, co ułatwia atakującemu odkrycie klucza i złamanie szyfru
  • Szyfr Polibiusza nie chroni przed atakami z użyciem technik kryptoanalitycznych, takich jak analiza częstotliwości trójliterowych słów

#include iostream
#include string
using namespace std;

string szyfrujPolibiusza(string wiadomosc) {
    string szyfr;
    char tabela[5][5] = {
        {'A', 'B', 'C', 'D', 'E'},
        {'F', 'G', 'H', 'I', 'K'},
        {'L', 'M', 'N', 'O', 'P'},
        {'Q', 'R', 'S', 'T', 'U'},
        {'V', 'W', 'X', 'Y', 'Z'}
    };
    for (int i = 0; i < wiadomosc.length(); i++) {
        char c = toupper(wiadomosc[i]);
        if (c >= 'A' && c <= 'Z') {
            for (int row = 0; row < 5; row++) {
                for (int col = 0; col < 5; col++) {
                    if (tabela[row][col] == c) {
                        szyfr += to_string(row+1) + to_string(col+1);
                    }
                }
            }
        }
    }
    return szyfr;
}
                        
string deszyfrujPolibiusza(string szyfr) {
    string wiadomosc;
    char tabela[5][5] = {
        {'A', 'B', 'C', 'D', 'E'},
        {'F', 'G', 'H', 'I', 'K'},
        {'L', 'M', 'N', 'O', 'P'},
        {'Q', 'R', 'S', 'T', 'U'},
        {'V', 'W', 'X', 'Y', 'Z'}
    };
    for (int i = 0; i < szyfr.length(); i += 2) {
        int row = szyfr[i] - '0' - 1;
        int col = szyfr[i+1] - '0' - 1;
        if (row >= 0 && row < 5 && col >= 0 && col < 5) {
            wiadomosc += tabela[row][col];
        }
    }
    return wiadomosc;
}
