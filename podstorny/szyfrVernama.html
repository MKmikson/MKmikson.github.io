Szyfr Vernama

Szyfr Vernama, znany również jako szyfr strumieniowy, to szyfr kryptograficzny, który generuje ciąg pseudolosowy bitów do zaszyfrowania tekstu jawnego. Każda litera tekstu jawnego jest szyfrowana za pomocą klucza, który składa się z ciągu bitów o takiej samej długości jak tekst jawnego. W skrócie, szyfr Vernama jest jednym z najbezpieczniejszych szyfrów, ponieważ generuje losowe klucze, które są używane tylko raz. Jednak, wymaga bezpiecznego generowania i przesyłania klucza oraz jest czasochłonny w procesie szyfrowania tekstu

Plusy szyfru Vernama:

  • Jest nieprzewidywalny i bezpieczny, ponieważ klucz jest generowany losowo i nie jest używany więcej niż raz
  • Zapewnia doskonałą poufność, ponieważ szyfrowany tekst jest jednorazowy, tzn. zaszyfrowany tekst nie może być odszyfrowany bez oryginalnego klucza
  • Nie ma ograniczeń co do długości tekstu jawnego, co oznacza, że szyfrowanie może być stosowane na dowolnie długich tekstach

Minusy szyfru Vernama:

  • Wymaga bezpiecznego i losowego generowania klucza, który musi być zawsze takiej samej długości jak tekst jawnego. Generowanie klucza może być trudne i kosztowne
  • Szyfrowanie tekstu jest czasochłonne, ponieważ każda litera jest szyfrowana oddzielnie i wymaga dużej ilości operacji matematycznych
  • Wymaga bezpiecznego przesyłania klucza, ponieważ jeśli klucz zostanie skradziony lub przechwycony, osoba trzecia będzie w stanie odszyfrować zaszyfrowany tekst

#include iostream
#include string
#include vector
#include random
#include ctime
using namespace std;
                            
// funkcja kodująca wiadomość przy użyciu klucza
string szyfrujVernama(string wiadomosc, string klucz) {
    string szyfrogram = "";
    for (int i = 0; i < wiadomosc.length(); i++) {

        char c = wiadomosc[i] ^ klucz[i];
        szyfrogram += c;
        }
        return szyfrogram;
    }
                        
// funkcja generująca losowy klucz o podanej długości
string generujKlucz(int dlugosc) {
    string klucz = "";
    // generator liczb losowych
    default_random_engine generator(time(0));
    uniform_int_distribution(int) distribution(0, 255);
    for (int i = 0; i < dlugosc; i++) {
        // generowanie kolejnego bajtu klucza
        int bajt = distribution(generator);
        klucz += (char)bajt;
    }
    return klucz;
}

    int main() {
    string wiadomosc = "Tajna wiadomość";
    string klucz = generujKlucz(wiadomosc.length()); 
    cout << "Wiadomość: " << wiadomosc << endl;
    cout << "Klucz: " << klucz << endl;
    string szyfrogram = szyfrujVernama(wiadomosc, klucz); 
    cout << "Zaszyfrowana wiadomość: " << szyfrogram << endl;
    string odszyfrowanaWiadomosc = szyfrujVernama(szyfrogram, klucz);
    cout << "Odszyfrowana wiadomość: " << odszyfrowanaWiadomosc << endl;
    return 0;
}
